# githubdashboard 

这是一个用flutter写的简易的Github客户端，它涵盖了包括登录，获取用户Event和相应的Github Repository及详情信息的功能，作为一个测试使用Flutter Framework的存在.  [English Version](https://github.com/underwindfall/FlutterGithubDashboard/blob/develop/README.md)

## 开始使用
